### 📌 2. Sales Prediction using Machine Learning

#### 📖 Overview

This project predicts product sales based on advertising budgets using a regression model.

#### 🎯 Objective

To analyze how advertising spending (TV, Radio, Newspaper) affects sales and build a predictive model.

#### 🛠️ Workflow

* Data preprocessing
* Correlation analysis
* Feature selection
* Model training using Linear Regression
* Model evaluation

#### 📊 Key Insights

* TV advertising has the strongest impact on sales
* Radio has moderate impact
* Newspaper has relatively low impact

#### 🤖 Model Used

* Linear Regression

#### 📈 Evaluation Metrics

* Mean Squared Error (MSE)
* R² Score

#### 🧰 Tools & Technologies

* Python
* Pandas, NumPy
* Seaborn, Matplotlib
* Scikit-learn
